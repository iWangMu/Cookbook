<, >, <=, >=, ==, !=



关系运算符的优先级低于算术运算符。例如，表达式 i + j < k – 1 意思是(i + j)< (k – 1)。关系运算符都是左结合的。

i < j < k ==》 (i < j) < k ==> 0/1 < k



逻辑运算符

&& || ！ 与或非

逻辑运算符所产生的结果是 0 或 1。

对操作数进行 “短路”计算

运算符!的优先级和一元正负号的优先级相同，运算符&&和运算符||的优先级低于关系运 算符和判等运算符。例如，表达式 i < j && k == m 等价于表达式(i < j) && (k == m)。 运算符!是右结合的，而运算符&&和运算符||都是左结合的。
